<h3>Юзер</h3>

| Стовпець   | Тип          | Обмеження        | Опис                            |
|------------|--------------|------------------|---------------------------------|
| id         | SERIAL       | PRIMARY KEY      | Ідентифікатор користувача       |
| email      | VARCHAR(255) | UNIQUE, NOT NULL | Email користувача               |
| tag        | VARCHAR(96)  | UNIQUE, NOT NULL | Тег                             |
| nickname   | VARCHAR(128) | NOT NULL         | Відображуване ім'я юзера        |
| bio        | TEXT         | NULLABLE         | Опис юзера                      |
| password   | VARCHAR(255) | NOT NULL         | Хешований пароль                |
| created_at | TIMESTAMP    | DEFAULT NOW()    | Час створення облікового запису |

<h3>Чат</h3>

| Стовпець    | Тип          | Обмеження     | Опис                            |
|-------------|--------------|---------------|---------------------------------|
| id          | SERIAL       | PRIMARY KEY   | Ідентифікатор користувача       |
| name        | VARCHAR(128) | NOT NULL      | Назва чату                      |
| tag         | VARCHAR(96)  | NOT NULL      | Тег чату                        |
| description | TEXT         | NULLABLE      | Опис чату                       |
| created_at  | TIMESTAMP    | DEFAULT NOW() | Час створення облікового запису |


<h3>Учасник групи</h3>

| Стовпець | Тип       | Обмеження   | Опис                      |
|----------|-----------|-------------|---------------------------|
| id       | SERIAL    | PRIMARY KEY | Ідентифікатор користувача |
| chatId   | FK        | NOT NULL    | ID чату                   |
| userId   | FK        | NOT NULL    | ID юзеру                  |
| role     | role enum | NOT NULL    | Роль                      |

<h3>Повідомлення</h3>

| Стовпець  | Тип    | Обмеження   | Опис                                  |
|-----------|--------|-------------|---------------------------------------|
| id        | SERIAL | PRIMARY KEY | Ідентифікатор користувача             |
| content   | TEXT   | NOT NULL    | текст повідомлення                    |
| creatorId | FK     | NULLABLE    | ID користувача який створив           |
| chatId    | FK     | NOT NULL    | ID чату в якому створене повідомлення |

<h3>Реакція</h3>

| Стовпець  | Тип    | Обмеження   | Опис                               |
|-----------|--------|-------------|------------------------------------|
| id        | SERIAL | PRIMARY KEY | Ідентифікатор користувача          |
| type      | FK     | NOT NULL    | тип реакції                        |
| messageId | FK     | NOT NULL    | повідомлення                       |
| ownerId   | FK     | NOT NULL    | учасник чату який поставив реакцію |

<h3>Тип реакції </h3>

| Стовпець | Тип          | Обмеження   | Опис                               |
|----------|--------------|-------------|------------------------------------|
| id       | SERIAL       | PRIMARY KEY | Ідентифікатор користувача          |
| name     | VARCHAR(128) | NOT NULL    | назва реакції                      |
| iconUrl  | FK           | NOT NULL    | посилання на іконку реакції        |
